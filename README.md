# Thai Language for Flarum


## Installation, Updating and Uninstallation

- Updating
```
composer require flarum-lang/japanese
```
- Updating
```
composer update flarum-lang/japanese
```
- Uninstallation
```
composer remove flarum-lang/japanese
```
